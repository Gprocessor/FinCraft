# Multi-tenant: one deployment, domain->tenant. Each tenant is a fully isolated Postgres database (not a shared table) — Fineract runs a full Liquibase migration for it. ./add-tenant.sh <id> <domain> "Name" (Fineract auto-restarts to migrate, registers the domain in TENANT_DOMAINS and regenerates config.js for you), add domain to DOMAINS (primary first), DNS to same IP, re-run ./setup-vm.sh for the cert.
